## Run away 기획 배경
1학기 동안 학습한 내용을 바탕으로 SpringBoot, Vue, MySQL과 한국관광공사 공공데이터를 이용해 지역별 여행지 정보를 쉽게 제공하기 위해 기획된 서비스입니다.
